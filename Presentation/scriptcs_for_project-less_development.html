What is it?

ScriptCs makes it easy to write and execute C# with a simple text editor

    
    

.Net scripting engines

  • Microsoft.Scripting and Dynamic Language Runtime: IronPython, IronRuby, …
  • Microsoft “Roslyn” CTP
  • CsScript

Install Chocolatey

C:\> @powershell -NoProfile -ExecutionPolicy unrestricted -Command "iex ((new-object net.webclient).DownloadString('https://chocolatey.org/install.ps1'))" && SET PATH=%PATH%;%systemdrive%\chocolatey\bin

C:\> cinst ScriptCs

ScriptCs advantages

  • Distributed under Apache 2 license
  • No project, just script
  • No IDE, any text editor
  • No compilation
  • Packages over assemblies
